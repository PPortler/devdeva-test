# 📊 Dashboard & Graph Test Project

โปรเจกต์สำหรับทดสอบระบบ Dashboard และ Graph Visualization ด้วย React + TypeScript

---

## 🚀 Overview

โปรเจกต์นี้มี 2 หน้าหลัก:

### 1. 📋 Dashboard
- แสดงข้อมูล Task
- Filter / Search / Status management
- UI แบบ Kanban (Todo / Doing / Done)

### 2. 📈 Graph Page
- แสดงกราฟ Multi-axis (Recharts)
- Toggle line chart ได้ (แสดง/ซ่อนแต่ละเส้น)
- Export กราฟเป็น PDF

---

## 🧰 Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Ant Design
- Zustand (State Management)
- Recharts (Chart)
- html2canvas (Export image)
- jsPDF (Export PDF)
- Lucide React (Icons)

---

## 📦 Install & Run

```bash
npm install      # install dependencies
npm run dev      # start dev server
npm run build    # build production
npm run lint     # check code quality
